## 💡 Infrastructure Philosophy

Every project in this roadmap runs on **$0/month** of infrastructure:

- **AWS Free Tier** → EC2 t2.micro, Lambda (1M req/month forever), DynamoDB (25GB forever)
- **Google Cloud Free** → e2-micro VM (forever), $300 credit
- **Cloudflare Free** → Workers, Pages, R2 (10GB), Tunnel, KV
- **LocalStack** → Full AWS simulation offline
- **Laptop (WSL2 + Docker)** → Primary compute for all local work
- **Minikube / kind** → Kubernetes clusters locally
- **All tools** → 100% open source

The goal: prove that serious cloud engineering skills can be built without spending money on infrastructure. The learning, not the budget, is the differentiator.
